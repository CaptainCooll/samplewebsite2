# Explore My World

A fun, game-like homepage where you explore a map to unlock different sections.

## How to Play

- Move your mouse around the map to explore.
- The character follows your mouse and reveals parts of the map.
- When you reveal a new area, a section (About, Team, Gallery, Events, Merch) will appear.
- Customize your character using the dropdown.

## Files

- `index.html`: The main HTML file.
- `style.css`: Styles for the page.
- `script.js`: JavaScript for the game logic.

## Running

Open `index.html` in your web browser to start exploring!
